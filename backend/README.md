# Habit Tracker - Backend (Semana 1)

Este proyecto es una aplicación para la gestión de hábitos basada en el libro "Hábitos Atómicos".

## Requisitos previos
- Node.js instalado
- Cuenta en MongoDB Atlas

## Configuración
1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Crear un archivo `.env` basado en el siguiente formato:
   ```env
   PORT=3000
   MONGODB_URI=tu_uri_de_mongodb_atlas
