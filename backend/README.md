# Habitos Backend

Proyecto backend desarrollado con Node.js, Express y MongoDB Atlas utilizando Mongoose.

## Requisitos
- Node.js instalado
- MongoDB Atlas configurado
- Archivo .env con la variable MONGO_URI

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:

npm install

## Ejecutar el proyecto

npm start

El servidor se ejecutará en:
http://localhost:3001

## Endpoints disponibles

GET /api/habits
POST /api/habits
PUT /api/habits/:id
DELETE /api/habits/:id

## Tecnologías utilizadas
- Node.js
- Express
- MongoDB Atlas
- Mongoose
